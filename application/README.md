# Application Guides

This directory contains the **application-focused guides** for the **CloudPedagogy AI Capability Framework (2026 Edition)**.

These documents translate the core framework into **practical, role-aware guidance** for applying AI capability in real organisational, educational, and professional contexts.

The materials here are designed to support **decision-making, implementation, and reflective practice** rather than theoretical exposition alone.

---

## What This Directory Contains

This directory includes the following application guides:

- **AI Capability Framework Application Handbook**  
  The comprehensive handbook for applying the AI Capability Framework across institutions, projects, and systems.

- **AI Capability Framework Application Handbook — Executive Summary**  
  A concise overview intended for senior leaders, decision-makers, and time-constrained readers.

- **AI Capability Framework — Quick Reference Guide for Course Designers**  
  A practical guide supporting educators and learning designers in applying the framework within course and programme design.

Each guide is written to stand alone while remaining conceptually aligned with the full framework.

---

## How These Guides Are Intended to Be Used

These application guides support:

- Institutional AI strategy and governance  
- Curriculum, assessment, and programme design  
- Professional development and CPD  
- Project-level AI decision-making  
- Ethical review, assurance, and reflective practice  

They can be used independently or alongside the core framework, self-assessment matrix, toolkits, and scenario packs.
