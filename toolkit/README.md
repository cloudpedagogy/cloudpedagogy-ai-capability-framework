# Toolkit

This folder contains the **core practical toolkits** for applying the **CloudPedagogy AI Capability Framework (2026 Edition)** in real professional, educational, research, and governance contexts.

The materials here translate the Framework from a conceptual reference model into **usable, facilitation-ready artefacts** that support reflection, ethical decision-making, governance, and capability development.

## Source Documents

Editable Word versions of each toolkit are provided in: source-docs/word/

These files are included intentionally to support adaptation, completion, and local use in professional and institutional contexts.

- The **Word documents** are the editable working versions, intended for adaptation, completion, and local use.
- The **Markdown files** are the published reference versions, intended for reading, citation, and version control.

Users may work with **either or both**, depending on their needs.
